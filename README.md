# Calculadora Web 
 
Proyecto desarrollado por **Sharick Olmos**
 
---
 
## ¿Qué es?
 
Una calculadora web sencilla hecha con HTML, CSS y JavaScript. Permite ingresar dos números, elegir una operación y ver el resultado.
 
## Funciones
 
- Suma, resta, multiplicación y división
- Botón para borrar el último dígito (←)
- Botón para limpiar todo (C)
- Muestra el primer número, la operación, el segundo número y el resultado en una fila
## Archivos
 
| Archivo | Descripción |
|---|---|
| `basica.html` | Estructura de la calculadora |
| `basica.css` | Estilos y colores |
| `basica.js` | Lógica y operaciones |
 
## ¿Cómo usarla?
 
1. Descarga los 3 archivos en una misma carpeta
2. Abre `basica.html` en tu navegador
3. Listo
## ¿Cómo funciona?
 
1. Presiona los números del primer valor
2. Presiona una operación ( + − × ÷ )
3. Presiona los números del segundo valor
4. Presiona **=** para ver el resultado
## Tecnologías usadas
 
- HTML5
- CSS3
- JavaScript
